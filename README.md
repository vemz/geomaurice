# Observatoire Résilience Côtière - Maurice

Ce projet est une carte interactive visualisant les données de résilience côtière pour l'Île Maurice. Il inclut des informations sur les poches de pauvreté, les zones inondables, les risques côtiers, l'énergie verte et l'historique des crues.

## Structure du Projet

- `index.html`: La page principale de l'application.
- `style.css`: Les styles pour la mise en page et le design.
- `app.js`: La logique JavaScript pour la carte (Leaflet) et les graphiques (Chart.js).
- `sources.md`: Les sources des données utilisées.

## Comment lancer le projet localement

Il suffit d'ouvrir le fichier `index.html` dans votre navigateur web.
